Alert

Los alerts son mensajes contextuales para información al usuario o para acciones que debe realizar. Reciben la clase alert para el formato y una clase alert-* donde el * es un color de Bootstrap.

Colores: primary, secondary, success, dange, warning, info, light, dark.
